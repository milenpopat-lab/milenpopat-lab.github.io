Financial Analytics Dashboard

Python Streamlit Plotly yfinance Pandas
Live Demo Code

Built an interactive web application that provides comprehensive stock market analysis and portfolio management tools. The dashboard fetches real-time financial data via the yfinance API and calculates key performance metrics including Sharpe ratios, Value at Risk (VaR), maximum drawdown, and correlation analysis. Features four analytical tabs covering market overview, individual stock performance with candlestick charts, portfolio-level analysis with equal-weight allocation, and comprehensive risk metrics.

The application showcases advanced Python programming with Pandas for data manipulation, NumPy for financial calculations, and Plotly for dynamic visualizations. Implemented caching mechanisms for optimal performance and deployed on Streamlit Cloud for public access. This project demonstrates proficiency in financial modeling, API integration, data visualization, and full-stack web application development.

Financial Statement Analysis Dashboard

Python Streamlit Financial Analysis Ratio Analysis yfinance
Live Demo Code

Developed a comprehensive financial statement analysis platform that automatically fetches and analyzes income statements, balance sheets, and cash flow statements for any publicly traded company. The dashboard calculates 13+ critical financial ratios across profitability (ROE, ROA, Net Margin), liquidity (Current Ratio, Quick Ratio), leverage (Debt-to-Equity), and efficiency metrics. Features include multi-year trend analysis, year-over-year growth calculations, and side-by-side peer comparison functionality for competitive benchmarking.

Implemented intelligent ratio interpretation with color-coded indicators for financial health assessment. The application provides visual representations through interactive charts, radar plots for ratio comparison, and waterfall diagrams for cash flow analysis. Users can analyze any public company and compare it with industry peers to identify competitive advantages and investment opportunities. This project demonstrates expertise in financial modeling, corporate finance analysis, and the ability to transform complex financial data into actionable insights for investment decision-making.

Basic Stock Prediction

Python Prophet Machine Learning Time Series Streamlit
Live Demo Code

Developed a machine learning application that forecasts future stock prices using Facebook's Prophet algorithm for time series analysis. The application allows users to select from multiple stocks (Google, Apple, Microsoft, GameStop) and predict price movements up to 4 years into the future. Implements Prophet's additive regression model to capture seasonality, trends, and holiday effects in stock price movements, providing both point estimates and confidence intervals for predictions.

Features interactive visualizations powered by Plotly that display historical price data with open/close trends, forecasted values with uncertainty bands, and decomposed forecast components showing trend and seasonal patterns. The application handles data preprocessing including duplicate removal, missing value treatment, and timestamp normalization to ensure robust model training. This project demonstrates practical application of machine learning for financial forecasting, understanding of time series modeling, and ability to build user-friendly interfaces for complex analytical tasks.
